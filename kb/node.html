# Windows

新版本的Windows默认的终端时PowerShell，在VSCode里面如果直接运行"npm install"等命令会提示 "因为在此系统上禁止运行脚本"的异常信息。

解决方法如下

# 1. 以管理员身份运行VSCode
# 2. 在终端里面执行下面命令，查看当前状态 
get-ExecutionPolicy
# 3. 执行下面命令修改策略 
set-ExecutionPolicy RemoteSigned
# 4. 以正常方式启动VSCode

# node-sass

# 卸载
npm uninstall node-sass -g
# 下载地址
https://github.com/sass/node-sass/releases/download/v4.14.1/win32-x64-83_binding.node
https://github.com/sass/node-sass/releases/download/v4.14.1/darwin-x64-83_binding.node

设置环境变量

set SASS_BINARY_PATH=D:\Tools\Node\node-sass\win32-x64-64_binding.node
echo %SASS_BINARY_PATH%
npm i -g node-sass
