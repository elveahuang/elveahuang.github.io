系统清理瘦身 ​

  1. 虚拟内存迁移，在"系统属性 - 高级 - 性能设置"中，将虚拟内存文件转移至其他磁盘，可释放 1-4GB 空间。
  2. 系统还原点管理，通过命令提示符输入“vssadmin delete shadows /all”，可删除所有还原点，释放 5-10GB 空间。
  3. 休眠文件关闭，在命令提示符输入“powercfg -h off”，可关闭休眠功能，释放与内存大小相同的空间。
  4. 自动保存文档，在 OneDrive 设置中关闭 "自动保存文档" 功能，可减少 C 盘写入量。
  5. 临时文件清理，在资源管理器地址栏输入%temp%，手动删除所有文件，平均释放 0.5-2GB 空间。
  6. 系统更新备份清理，通过"设置 - 系统 - 存储 - 临时文件"，删除 Windows 更新备份，平均释放 5-8GB 空间。
