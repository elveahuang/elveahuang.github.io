安装 ​

Windows ​

设置环境变量 ​

shell
setx BUN_HOME "D:\Tools\Node\bun"
1

RHEL ​

bash
#
mkdir -p /data/tools/node/bun
# 解压缩
tar -Jxvf bun-linux-x64.zip
mv ./bun-linux-x64/** /data/tools/node/bun
rm -rf ./bun-linux-x64
1
2
3
4
5
6

添加环境变量

bash
vi /etc/profile.d/bun.sh
1

添加一下内容

bash
# 环境变量
export BUN_HOME=/data/tools/node/bun
export PATH=$PATH:$BUN_HOME/bin:.
1
2
3

启用环境变量

bash
source /etc/profile
1

检查是否成功安装

bash
bun -v
