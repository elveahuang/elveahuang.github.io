安装部署 - RHEL ​

先安装ErLang

bash
sudo yum install -y erlang-28.0.2-1.el9.x86_64.rpm
sudo yum install -y erlang-28.0.2-1.el8.x86_64.rpm
sudo yum install -y erlang-28.0.2-1.el7.x86_64.rpm

bash
sudo yum install -y rabbitmq-server-4.1.1-1.el8.noarch.rpm

sudo systemctl restart rabbitmq-server.service
# 关闭服务
sudo systemctl stop rabbitmq-server.service
# 查看状态
sudo systemctl status rabbitmq-server.service

WSL ​

默认情况下，RabbitMQ在WSL里面只绑定了IPv6地址，这样子导致无法连接，所以需要在配置文件里面增加配置项来绑定IPv4地址。

bash
sudo vi /etc/rabbitmq/rabbitmq.conf
1

增加以下内容

bash
listeners.tcp.default=127.0.0.1:5672
1

安装部署 - Windows ​

RabbitMQ依赖于Erlang，所以需要先安装Erlang。

ErLang官网

ErLang下载地址

安装目录 ​

bash
|---D                                       
|---|---Tools                                 -- 
|---|---|---RabbitMQ                          -- RabbitMQ安装目录
|---|---|---ErLang                            -- ErLang安装目录
1
2
3
4

把ErLang和RabbitMQ安装到对应的目录下，并设置好环境变量

bash
%ERLANG_HOME%=D:\Tools\RabbitMQ\Erlang\erts-15.2.6
%ERLANG_SERVICE_MANAGER_PATH%=D:\Tools\RabbitMQ\Erlang\erts-15.2.6\bin
%RABBITMQ_HOME%=D:\Tools\RabbitMQ\Server
1
2
3

静默安装

bash
#
otp_win64_27.3.3.exe /S /D=D:\Tools\RabbitMQ\Erlang
#
tar -xzvf rabbitmq-server-windows-4.1.0.zip -C D:\Tools\RabbitMQ
