LibreOffice ​

安装部署 ​

RHEL ​

解压缩

bash
tar zxvf LibreOffice_25.2.2_Linux_x86-64_rpm.tar.gz
tar zxvf LibreOffice_25.2.2_Linux_x86-64_rpm_langpack_zh-CN.tar.gz
1
2

安装

bash
yum install -y ./LibreOffice_25.2.2.2_Linux_x86-64_rpm/RPMS/*.rpm
yum install -y ./LibreOffice_25.2.2.2_Linux_x86-64_rpm_langpack_zh-CN/RPMS/*.rpm
1
2

检查是否安装成功

bash
ldd /opt/libreoffice25.2/program/soffice.bin
1

如果有依赖缺失，那补充安装依赖项并重新检查

bash
yum install -y cairo
yum install -y libSM
yum install -y ibus
yum install -y libX11-xcb
1
2
3
4

新建软连接，以支持 JodConverter。

bash
ln -s /opt/libreoffice7.6 /opt/libreoffice
1

卸载

bash
yum erase libreoffice\*
1
