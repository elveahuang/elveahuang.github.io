# Node.js

# 安装

Node.js的下载地址，推荐绿色安装包。

安装目录如下：

|---node
    |---node            （主安装目录）
    |---node-npm        （全局安装的模块目录）
    |---node-cache      （缓存目录）
    |---node-sass       （node-sass二进制文件保存目录）

# MacOS

# 设置缓存目录
npm config set cache "/Users/elvea/Tools/node/node-cache"
# 设置全局模块目录
npm config set prefix "/Users/elvea/Tools/node/node-npm"

# Windows

# 设置缓存目录
npm config set cache "D:\Dev\Node\node-cache"
# 设置全局模块目录
npm config set prefix "D:\Dev\Node\node-npm"

# 设置仓库地址

# 设置阿里镜像
npm config set registry="https://registry.npm.taobao.org"
# 还原官方镜像
npm config set registry="https://registry.npmjs.org"

# node-sass

# 正确的安装姿势

移除卸载本地已经安装的模块

npm uninstall node-sass -g

下载官方的模块文件

https://github.com/sass/node-sass/releases/download/v4.9.4/darwin-x64-64_binding.node
https://github.com/sass/node-sass/releases/download/v4.9.4/darwin-x64-64_binding.node

设置环境变量

#
set SASS_BINARY_PATH=D:\Dev\Node\node-sass\win32-x64-64_binding.node
#
echo SASS_BINARY_PATH

安装node-sass

npm i -g node-sass
